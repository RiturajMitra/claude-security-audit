## 📋 What is claude-security-audit?

claude-security-audit is an application designed to help you check your website or system security. It scans for common issues like weak passwords, missing security headers, and known software weaknesses. The audit covers popular security risks listed under OWASP Top 10, and checks for vulnerabilities identified by CWE and CVE standards. It also reviews your authentication setup, paywall protection, and infrastructure safety.

This tool helps you understand where your system might be at risk and what to fix. You do not need to be a technical expert to use it. The program runs on Windows and provides clear results for you to follow.

## ⚙ Common Questions and Troubleshooting

### Why doesn’t the installer run?

Make sure Windows Defender or other antivirus software did not block it. Also, confirm that the file fully downloaded.

### The scan stops before finishing. What now?

Check your internet connection and firewall settings. Restart claude-security-audit with administrator rights and try again.

### How do I interpret the risk levels?

- **High Risk** means immediate attention is needed to fix a security flaw.  
- **Medium Risk** indicates areas that could cause issues but are not urgent.  
- **Low Risk** shows minor advice or best practice suggestions.

### Can I use this tool for websites only?

While it is built mainly for website audits, you can check other networked systems as long as you provide their address.
